• Description

  • This is a UI Theme that was designed to work along with Lineal Music Controls. It themes tabs of multiple apps as shown in the screenshots. It was tested on @3x devices such as iPhone XS (iOS 12.1.1) and iPhone X (iOS 11.3.1).

    @2x devices were included, I just don't have any devices to see how it looks on them.

    Important Notes:

    • This needs a theming engine which supports UI theming. As of 2019-5-29, only iThemer can accomplish this. Please don't ask about other engines.
    • Included as well are images for SafariPlus Downloads button. You can copy/move them to /Library/Application Support/SafariPlus.bundle and replace the images there (back them up first)

    Lastly, this is free. Donations are always welcome though if you find that you like my work.

Description

Designed to work with Flat Music Controls.

It’s made for @3x and @2x devices, but I only managed to test it on iPhone X (iOS 11.3.1) and iPhone XS (iOS 12).

Themes apps and UIs such as:

  • Safari and WebView
  • Phone
  • Music
  • Video Player
  • Sileo
  • Some Sharesheet Icons such as Copy, Print, Add to Favorites, Add to Bookmarks, Create PDF
  • App Store

Has an additional option of having smaller navigation arrows (they look nicer if you have CarPlay)

SafariPlus

you will find the download icons inside the theme folder by going to the path /Library/Themes/Flat UI.theme/bundles/SafariPlus.bundle.

You can copy/move them to /Library/Application Support/SafariPlus.bundle and replace the images there (back them up first).

Important Notes

As of 2019-6-5, this only works with iThemer. Please don’t ask me about Snowboard or Anemone. It’s up to the developers to implement UI theming, not me.
